# Receiving user flow

1. Open receiving task.
2. Scan supplier delivery document.
3. Validate supplier, material, purchase reference, and expected quantity.
4. Scan container or generate container label.
5. Capture gross/net weight.
6. Create pending-inspection batch.
7. Print label and show submission status.
8. If offline, store one idempotent command and display it in the offline queue.
