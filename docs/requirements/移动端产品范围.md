# 移动端产品范围

## 1. 产品定位

`mom-mobile` 是工业 MOM 的现场作业 PDA 客户端，服务于仓库、生产和质量现场的扫码型任务。

它不是管理后台的移动版，也不承担复杂报表、主数据维护或跨工厂配置。

## 2. V1 目标

V1 以电解液制造为主演示场景，完成：

```text
送货任务
→ 原料收货
→ 来料检验交接
→ 原料上架
→ 生产领料
→ 生产退料
→ 成品入库
→ 发运装车
→ 离线同步与冲突恢复
```

## 3. 核心用户价值

- 通过扫码减少手工输入和选错对象。
- 通过任务列表减少现场人员理解系统模块的成本。
- 通过离线命令避免网络短时中断导致作业丢失。
- 通过幂等与冲突处理避免重复收货、重复扣减和重复发运。
- 通过明确的视觉、声音和振动反馈快速识别操作结果。

## 4. V1 包含范围

### 4.1 首页和任务

- 当前用户、工厂和网络状态。
- 按角色展示待办任务。
- 最近任务和离线命令入口。
- 设备、版本和诊断信息入口。

### 4.2 原料收货

- 扫描供应商送货单。
- 校验供应商、采购参考、物料和预计数量。
- 扫描或生成容器编号。
- 输入毛重、皮重或净重。
- 建立待检批次。
- 显示标签打印结果或待打印状态。

### 4.3 来料检验交接

- 查看待检批次。
- 扫描批次或容器确认交接。
- 展示检验状态，不在 PDA 上实现复杂检验录入。

### 4.4 上架

- 扫描容器。
- 扫描目标库位。
- 校验物料、质量状态、库区和库位兼容性。
- 提交上架确认。

### 4.5 生产领料与退料

- 扫描生产工单。
- 扫描物料、容器和批次。
- 校验工单需求、可用状态和数量。
- 提交领料。
- 退料时记录原因、数量和目标容器。

### 4.6 成品入库

- 扫描成品批次和容器。
- 展示质量放行状态。
- 扫描目标库位并提交入库。

### 4.7 发运确认

- 扫描发运单。
- 扫描托盘、容器或批次。
- 校验客户、产品、数量和放行状态。
- 完成装车或发运确认。

### 4.8 离线与恢复

- 展示网络状态。
- 持久化符合离线条件的业务命令。
- 展示待同步、同步中、失败、冲突和人工处理状态。
- 支持查看命令详情、重试、取消或转人工。

## 5. 非 V1 范围

- 复杂管理报表。
- 主数据维护。
- 完整质量检验工作站。
- 生产排程和工艺配置。
- 批次谱系图可视化。
- 设备参数下发。
- iOS 产品发布。
- 所有厂商 PDA SDK 同时适配。
- 后台常驻无限重试。
- 在设备本地保存完整业务数据库。

## 6. 系统边界

- 业务权威数据位于 MOM 后端。
- 浏览器或 App 只访问 MOM Gateway。
- Android PDA 负责采集现场操作和展示任务状态。
- 服务端负责权限、幂等、业务规则、库存和质量状态判定。
- 本地离线队列不是业务权威账本。

## 7. 当前实现状态

已具备：

- uni-app Vue 3 应用骨架。
- 首页、收货、上架、领料、发运和离线队列页面骨架。
- 扫码、网络和存储适配器。
- API Client 和请求标识生成。
- 基础离线命令入队、读取、删除和手工重试。
- H5 CI 构建。

尚未闭环：

- OAuth 登录。
- 正式 Gateway 契约。
- 自动同步 Worker。
- 冲突快照和处置。
- 真机厂商扫描 SDK。
- Android 签名和发布。
