# ADR-007：本地数据最小化并按用户与工厂隔离

- 状态：Accepted
- 日期：2026-07-18

## 背景

PDA 需要保存离线命令和少量恢复数据，但设备可能丢失、多人共用或在多个工厂切换。

## 决策

本地只保存作业和恢复必需的数据，并按环境、用户和工厂隔离。敏感数据优先不落盘；确需落盘时使用平台安全存储或经评审的加密方案。

## 允许数据

- 离线命令及状态。
- 非敏感任务摘要。
- 页面恢复草稿。
- 设备和应用诊断信息。

## 禁止数据

- 密码。
- 签名私钥。
- 完整 Token 日志。
- 完整业务数据库镜像。
- 非必要的客户和个人敏感数据。

## 理由

- 降低设备丢失影响。
- 避免账号和工厂串数据。
- 控制存储容量和迁移风险。

## 后果

- 需要 Storage Adapter 和 Key 规范。
- 需要数据过期、清理和升级迁移。
- 退出登录不能简单删除仍需处理的离线命令。

## 约束

- 离线命令记录创建用户和工厂。
- 新账号不能自动执行旧账号命令。
- 加密密钥不得硬编码在包中。

## 验证

- 用户和工厂数据隔离。
- 退出后 Token 清除，命令仍可受控处理。
- 存储和日志中无敏感凭证。
