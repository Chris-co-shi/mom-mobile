# 认证、权限与工厂范围

## 1. 认证目标

移动端使用 OAuth2.1/OIDC 兼容认证流程，并通过 MOM Gateway 访问后端。

## 2. 客户端类型

PDA 应拥有独立 OAuth Client，不与管理端 Web 共用 Client 配置。

需要单独定义：

- Redirect URI。
- Token 生命周期。
- Refresh 策略。
- 设备丢失后的撤销方式。
- App 链接或自定义 Scheme。

## 3. Token 管理

- Token 不写入普通日志。
- Token 存储应使用平台安全能力或最小化持久化。
- Access Token 过期时按统一流程刷新。
- Refresh 失败或被撤销时重新登录。
- 退出和账号切换时清理会话数据。

## 4. 权限层级

### 4.1 页面权限

控制是否可进入收货、上架、领料、退料、入库和发运页面。

### 4.2 操作权限

控制是否可提交、取消、重试、转人工或执行主管操作。

### 4.3 数据范围

控制可访问的：

- 集团。
- 工厂。
- 仓库。
- 库区。
- 业务组织。

客户端可以隐藏不可用操作，但最终授权必须由服务端执行。

## 5. 工厂上下文

- 登录后选择或确认当前工厂。
- 请求统一携带工厂上下文。
- 离线命令记录创建时工厂。
- 同步前重新校验用户是否仍有该工厂权限。
- 切换工厂时不得自动执行原工厂待处理命令。

## 6. 账号切换与离线命令

旧账号创建的命令不能由新账号 Token 静默执行。

处理方式：

- 提示原用户登录处理。
- 由具备权限的主管执行显式转交。
- 取消命令并重新创建。

转交必须留下审计信息。

## 7. 会话过期

页面应：

1. 停止新的写请求。
2. 保留尚未提交的可恢复输入。
3. 不删除已持久化离线命令。
4. 引导登录。
5. 登录后重新校验页面和命令权限。

## 8. 无权限状态

无权限页面需区分：

- 无页面权限。
- 无操作权限。
- 无当前工厂权限。
- 数据已转移到其他组织。
- 会话已过期。

## 9. 设备丢失

需支持：

- 撤销 Refresh Token。
- 禁用设备会话。
- 远程设备管理策略（后续）。
- 本地会话过期。
- 不在本地保存长期可复用凭证。

## 10. 验收

- 退出后旧 Token 不再可用。
- 切换工厂后请求上下文正确。
- 无权限按钮不显示，直接调用仍被服务端拒绝。
- 切换用户不执行旧用户命令。
- 会话过期后未提交输入可受控恢复。
